# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in Yggdrasil Forge, please report it privately:

- **Email:** (to be configured)
- **Do not** open a public issue.

We will respond within 7 days and aim to publish a fix within 90 days, depending on severity.
